"All the RL that's fit to read" — 大语言模型强化学习每日速览

编者按 · 今日要闻

今日收录 17 篇 全新 LLM 强化学习相关论文，主题呈现三大聚焦： (1) 策略优化算法的"后 GRPO 时代"分化 — EVPO（2604.19485）把 PPO 与 GRPO 的 baseline 选择重新表述为 Kalman 滤波问题，并用 Explained Variance 作为一键可算指标，在单步之内自适应切换 critic/batch-mean 两种优势估计，在经典控制、Agentic 与数学四任务上全面超越 PPO 与 GRPO； CalibAdv / Negative Advantage（2604.18235）首次量化 GRPO 在"深搜"任务中负优势的双刃剑效应，提出用校准项避免正确中间步被错判惩罚； LVLM 重思（2604.19857）构造 Tool-Augmented MDP 理论框架，为 Visual-ARFT 的收敛性与跨域泛化首次给出数学解释。

(2) Agentic RL 的"小模型 × 开放数据"路线成熟 — LiteResearcher-4B（2604.17931）构建合成"虚拟 Web"绕开真实搜索开销，在 GAIA/Xbench 上追平 Claude-4.5-Sonnet / GLM-4.6； DR-Venus-4B（2604.19859）在仅 10K 开放数据下证明 Agentic SFT → Agentic RL 二阶段配方对边缘部署的有效性； CoSearch（2604.17555）用语义聚类子查询把"推理+排序"放进同一 GRPO 群组，七个 QA 基准一致提升； OThink-SRR1（2604.19766）提出 Refine 证据蒸馏 阶段 + GRPO-IR 端到端算法，降 token 用量而提 QA 准确率。

(3) 奖励设计与工具行为的深度再审视 — R2IF（2604.20316）首次把 CoT 有效性量化为 CER 奖励并与 SMV（规范-修改-赋值）联合，在 BFCL 上 Llama3.2-3B +34.62%； ToolsRL（2604.19945）用"工具监督 → 准确率"二阶段课程避免异质任务冲突； POP（2604.20051）把 Self-Play 首次扩展到开放域，借"特权预训练文本"防 reward hacking； JTPRO（2604.19821）以 rollout 反思联合优化 prompt 与工具 schema； Tool-Overuse Illusion（2604.19749）揭示 LLM 对自身知识边界的"认知幻觉"，用 DPO 减少 82.8% 不必要调用。 此外，TRN-R1-Zero（2604.19070）将 GRPO 拓展至文本富网络零样本图推理，NGC（2604.18002）以结果奖励让模型在推理同时学会遗忘 KV 缓存，SaFeR-Steer（2604.16358）用"导师—学生" GRPO 循环完成多轮安全对齐，DCF（2604.20098）让 Conformal Factuality 首次成为可微训练目标，真实保留 +141%。

总体上，今日工作显示：策略优化从 GRPO 的单一形态分化为"自适应 / 校准 / 理论化"三种方向，Agentic RL 正迅速收敛到"开放数据 + 小模型 + 合成环境"的可复现配方，而奖励设计越过了"format+accuracy"阶段，开始量化 CoT、工具使用、遗忘、认知幻觉等细粒度行为指标。

I. 策略优化 · Policy Optimization

EVPO：用 Explained Variance 自适应切换 critic 与 batch-mean 基线

Chengjun Pan, Shichun Liu, Jiahang Lin, Dingwei Zhu, Jiazheng Zhang, Shihan Dou, Songyang Gao, Zhenhua Han, Binghai Wang, Rui Zheng, Xuanjing Huang, Tao Gui, Yansong Feng（北京大学 / 复旦大学 / 上海启计智丰 / 上海人工智能实验室）
arXiv: 2604.19485 · 2026-04-21

核心贡献：把 "PPO 带 critic" 与 "GRPO 用 batch 均值" 重新表述为同一 Kalman 滤波问题的两个极端，证明可解释方差 (Explained Variance, EV) 是唯一分界：EV>0 时 critic 减方差，EV≤0 时 critic 反而注入噪声超过信号。由此提出 EVPO——在每一步训练中用单批数据计算 EV，自适应选择最优 baseline。

方法：(i) 把优势估计视为对真实 Q 值的"带噪观测组合"，推导 Kalman 增益闭式解；(ii) 用当前 batch 的 critic 预测 R² 作为 EV，设置阈值 τ=0 做开关；(iii) 在稀疏奖励的 agentic 任务里，EVPO 能自动退化为 GRPO，在稠密奖励的数学任务里又能退化为 PPO。

关键结果：覆盖 Sokoban、FrozenLake、WebShop、MATH 四个任务，EVPO 一致超越 PPO 与 GRPO——在 Sokoban 稀奖励设置下比 PPO +3.6pp，同时比 GRPO +2.1pp；在 MATH 上比 GRPO +1.8pp。证明"是否使用 critic"不应是设计者先验，而是训练过程中动态可学的信号。

意义：PPO vs GRPO 的旷日争论迎来首个统一解释，也是 LLM RL 工具箱里第一个"零额外成本"的自适应基线。

Negative Advantage Is a Double-Edged Sword：为深搜任务校准 GRPO 优势

Jiayi Wu 等（8 名作者）
arXiv: 2604.18235 · 2026-04-19

核心贡献：首次量化揭示 GRPO 在"深搜 (deep search)"多轮 Agent 训练中两大失败模式： (1) 中间步正确但最终答案错误时，负优势会不加区分地惩罚所有 token，导致大量有价值的搜索轨迹被污染； (2) 在训练后期自然语言能力出现灾难性坍塌。

方法：提出 CalibAdv——为深搜任务设计的优势校准机制：对 group 内负优势 trajectory 进行前缀共享度加权，保留与正 trajectory 共享前缀的 token 的梯度符号，只惩罚真正"分叉错误"的 token；同时引入 language quality guardrail 监测自然语言熵。

结果：在 HotpotQA、Musique、2WikiMHQA 等多跳检索基准上相对基线 GRPO 平均提升 +4.7pp F1，同时消除训练崩溃。

意义：与 MCPO (2604.16972)、SPS (2604.16995) 形成"GRPO 负梯度三部曲"：MCPO 修掉负优势在已掌握提示上的消除作用，SPS 修掉负优势引起的概率挤压，CalibAdv 修掉负优势在多轮工具轨迹上的前缀误伤。

LVLM 中 RLVR 的再思考：收敛、奖励分解与泛化

作者信息待公开
arXiv: 2604.19857 · 2026-04-21

核心贡献：给 Visual-ARFT、MM-Eureka 等视觉 Agentic RL 训练首次提供理论地基：引入 Tool-Augmented MDP (TA-MDP) ——带有有界深度工具调用的多模态 MDP 形式化，在此框架内解决两个长期悬而未决的问题： (i) verifiable reward 的三分量结构（format、accuracy、tool-executability）如何影响 GRPO 收敛？ (ii) 为何仅用少量工具相关任务训练即可跨域泛化？

主要结果：(1) 证明三奖励项满足势函数可分解条件，GRPO 在 TA-MDP 上收敛速率取决于最慢的那一分量；(2) 给出跨域泛化的上界：与"工具调用分布的覆盖熵"成反比，这解释了为何"小工具集覆盖即可获得广泛迁移"。

意义：补上了多模态 Agentic RL 缺失的理论层，预计会成为未来 LVLM-RL 收敛性分析的标准工具。

OLLM：把"选项"作为显式潜变量嵌入 LLM，为强化学习提供结构化探索空间

Shashank Sharma, Janina Hoffmann, Vinay Namboodiri（University of Bath）
arXiv: 2604.19087 · 2026-04-21

核心贡献：提出 Options-based LLM (OLLM)——在输出 head 前插入 encoder+decoder 两层轻量"插件"，把传统的"下一个 token 的单一分布"替换为"由离散潜变量索引的一组候选 next-token options"。变化被显式建模，而不再依赖 temperature/top-p 等 post-hoc 采样启发式。

RL 视角：在 1.7B OpenMathReasoning 模型上，把 options 作为策略空间进行 RL 训练时无需 KL 惩罚——因为探索已被潜变量结构约束，策略方差显著降低。这挑战了"KL 是 LLM RL 必要正则"的主流假设。

结果：在 OmniMath 基准上，OLLM 相较 top-p 采样基线同时提升 token-level 准确率和 final-answer 正确率，且 RL 训练中的策略方差下降 ~40%。

意义：为"latent reasoning（潜状态推理）"方向（见 2604.15726）提供了一个可训练、可 RL 的具体实例化载体。

II. Agentic RL · 智能体强化学习

LiteResearcher：用合成虚拟 Web 替代真实搜索，4B 小模型比肩 Claude 4.5-Sonnet

Wanli Li, Bince Qu, Bo Pan, Jianyu Zhang, Zheng Liu, Pan Zhang, Wei Chen, Bo Zhang（浙江大学 / Simplex AI / 香港理工大学）
arXiv: 2604.17931 · 2026-04-20

核心贡献：针对"深度研究 Agent"的两大痛点——手工合成数据无法激发真实搜索能力、而在线真实搜索训练又不稳定且成本高——提出 LiteResearcher：构建一个镜像真实 Web 架构的虚拟世界，分三步合成真实搜索数据、模拟搜索环境、再经课程 RL 持续提升。

方法亮点：(i) 用 crawler + LLM 蒸馏把 100M 真实网页压缩为离线可重现的搜索图；(ii) Agent 的 RL 奖励包括"检索相关度 × 推理链长度 × 最终答案正确率"三分量；(iii) 从"简单多跳 → 开放式调查"递进 3-level 课程。

关键结果：LiteResearcher-4B 在 GAIA 上 71.3%、Xbench 上 78.0%，超过参数量高达 8× 的开源 Agent，追平甚至超越 Claude-4.5-Sonnet 与 GLM-4.6。相比同尺寸的 AgentCPM-Explore-4B（在线 Web 训练），GAIA +7.4pp / Xbench +8.0pp。

意义：证明"离线虚拟 Web + 合成真实搜索数据"可以取代昂贵的在线 RL，为深研 Agent 的社区复现提供第一个完整开源配方。

DR-Venus：仅用 10K 开放数据，训练 4B 边缘级深度研究 Agent

Venus Team：Sunhao Dai, Yong Deng, Jinzhen Lin, Yusheng Song, Guoqing Wang, Xiaofeng Wu, Yuqi Zhou, Shuo Yang, Zhenzhe Ying, Zhanwei Zhang, Changhua Meng, Weiqiang Wang（inclusionAI）
arXiv: 2604.19859 · 2026-04-21

核心贡献：研究"在开放数据受限下如何训练强大的小型 Deep-Research Agent"。提出两阶段配方：Agentic SFT（建立基础调用能力）+ Agentic RL（针对长时界任务的执行可靠性）。

方法：(i) 通过数据质量滤波和数据利用率优化把 10K 开放样本用到极致；(ii) RL 阶段用 outcome-level verifiable reward + trace-consistency 辅助信号；(iii) 开源训练框架 & 数据。

结果：DR-Venus-4B 在 BrowseComp / SimpleDR / xBench-DR 多项基准上刷新边缘部署 SOTA，与 LiteResearcher-4B（2604.17931）形成"合成虚拟 Web vs 少量真实数据"两条互补路线。

意义：为"成本 / 延迟 / 隐私敏感"的真实落地场景提供可复现范式，证明深研 Agent 不再是百亿级专属。

CoSearch：通过语义聚类把"推理+文档排序"联合塞进 GRPO 群组

Hansi Zeng, Liam Collins, Bhuvesh Kumar, Neil Shah, Hamed Zamani
arXiv: 2604.17555 · 2026-04-19（v2: 2026-04-21）

核心贡献：先以探针实验证明"固定检索器"与"oracle 检索"之间存在高达 +26.8% F1 的可利用差距，随即提出 CoSearch —— 用 GRPO 同时训练多步推理 Agent 与生成式文档排序器。

关键技巧：(i) 语义聚类策略——将 token 相似的子查询聚到同一 GRPO 组，无需额外 rollout 就能形成有效的 group-relative 优化；(ii) 复合奖励——排序质量信号 + trajectory-level outcome 反馈；(iii) 推理 Agent 与排序器共享一个 LLM backbone，参数解耦仅在输出头。

结果：在 7 个单跳/多跳 QA 基准上一致超越强基线；消融实验证实语义聚类是 +3.1pp 的主要来源。

意义：打破"检索器 = 黑箱工具"的默认假设，开启"推理–检索联合优化"新方向。

OThink-SRR1：Search–Refine–Reason 三阶段 + GRPO-IR 端到端 RAG 训练

Haijian Liang, Zenghao Niu, Junjie Wu, Changwang Zhang, Wangchunshu Zhou, Jun Wang
arXiv: 2604.19766 · 2026-04-21

核心贡献：指出动态检索 RAG 的两大瓶颈：(1) 无关检索噪声会误导推理；(2) 处理全文档代价高昂。提出 OThink-SRR1——在推理前增设 Refine 阶段，把检索到的文档蒸馏为简洁相关事实，再交给 Reason 阶段。

算法：提出 GRPO-IR（Information-Refined GRPO）——奖励同时考虑"关键证据识别准确率"与"过度检索惩罚"，实现端到端训练：Search 不过度、Refine 精准、Reason 高效。

结果：在 4 个多跳 QA 基准上准确率超越强基线，而所需检索步骤与 token 数量显著更少。

意义：与 AutoSearch (2604.17337) 的"最小充分深度"形成互补：前者压缩深度、后者压缩每步内容，共同推动"高效而准确"的 Agentic RAG。

SaFeR-Steer：多轮 MLLM 对齐的"导师-学生" GRPO 循环

Haolong Hu, Hanyu Li, Tiancheng He, Huahui Yi, An Zhang, Qiankun Li, Kun Wang, Yang Liu, Zhigang Zeng
arXiv: 2604.16358 · 2026-04-18

核心贡献：针对多轮 MLLM 安全对齐中常见的"一次性静态对抗数据"局限，提出 SaFeR-Steer：分阶段合成引导 + Tutor-in-the-Loop GRPO。在 RL 阶段一个统一 Safety Tutor 同时承担自适应后续攻击与安全性评分两职，让单个学生模型在持续演化的攻击下对齐。

方法：(i) Stage-1 合成引导：从弱对抗到强对抗逐步暴露学生弱点；(ii) Stage-2 GRPO：Tutor 根据 feedback dynamics 动态生成 follow-up 攻击，学生用 group-relative 比较学习最安全响应。

意义：把多轮对抗攻防从 static red-team 升级到 on-policy 攻防博弈，与 ARES（2604.18789）形成"联合攻 Policy+RM vs 单模型防"的两极。

ToolsRL：用工具监督打破多模态视觉推理的异质任务优化冲突

Qihua Dong（Northeastern University / Amazon AGI 实习）, Gozde Sahin, Pei Wang, Zhaowei Cai, Robik Shrestha, Hao Yang, Davide Modolo（Amazon AGI）
arXiv: 2604.19945 · 2026-04-21

核心贡献：针对多模态工具使用中"精度奖励与格式奖励相互冲突"的问题，提出 Tool-supervised RL (ToolsRL)：把工具监督直接作为显式中间奖励。只选择简单、原生、可解释的视觉工具——zoom-in、rotate、flip、draw point/line——这些工具的监督信号极易收集。

二阶段课程：Stage-1 仅用工具专项奖励（正确调用与否）训练；Stage-2 切换为准确率导向奖励，允许自由工具调用。两阶段分离避免了异质目标下的梯度冲突。

意义：补上了 Visual-ARFT、OpenVLThinker 等方法缺失的"工具使用监督"拼图，与 R2IF（2604.20316）在文本工具领域呼应。

JTPRO：联合反思优化 prompt + 工具 schema，提升工具调用可靠性

作者信息待公开
arXiv: 2604.19821 · 2026-04-21

核心贡献：发现大量工具调用失败源于两类问题：(i) 通用 prompt 忽略工具特性；(ii) 工具 schema 本身描述不清。提出 JTPRO (Joint Tool-Prompt Reflective Optimization)——用 rollout-driven 反思联合优化全局指令 + 每工具 schema/参数描述。

评估：在多工具基准上引入三指标 Tool Selection Accuracy (TSA)、Slot Filling Accuracy (SFA)、Overall Success Rate (OSR)。

结果：OSR 相对强基线一致提升 5%–20%。与 GEPA 一脉相承，但首次把工具描述本身列入联合优化变量。

Tool-Overuse Illusion：LLM 为何偏爱工具而轻视自身知识？

Yirong Zeng, Shen You, Yufei Liu, Qunyao Du, Xiao Ding, Yutai Hou, Yuxian Wang, Wu Ning, Haonan Song, Dandan Tu, Bibo Cai, Ting Liu
arXiv: 2604.19749 · 2026-04-21

核心发现：揭示"工具过度使用"是当前 LLM Agent 普遍存在的现象。通过跨不同内部知识可用区间的行为分析，归因于一个认知幻觉 (knowledge epistemic illusion)——模型系统性错判自身知识边界，无法感知"哪些问题自己已经知道"。

缓解方法：提出基于 DPO 的知识感知认知边界对齐策略：构造"应当 / 不应调用"的偏好对，直接优化 Agent 在知识可用区间的自信度。

结果：工具使用减少 82.8%，同时准确率不降反升。证明"少调用 = 更诚实"可以是明确的对齐目标。

意义：给"工具调用效率"第一次赋予了认知论意义，不再只是 cost/latency 问题。

III. 推理 RL · Reasoning RL

TRN-R1-Zero：用纯 RL 让 LLM 掌握文本富网络 (TRN) 推理

作者信息待公开
arXiv: 2604.19070 · 2026-04-21

核心贡献：零样本 TRN 推理仍是前沿难题——需要同时整合文本语义与关系结构。提出 TRN-R1-Zero：一个完全依靠 RL 的 post-training 框架，无需任何 SFT 或来自大型推理模型的 CoT 数据。

算法：提出 Neighbour-aware GRPO——基于一个全新的边际增益 (margin gain) 指标动态调整奖励，该指标量化邻居信号的信息量，从而显式引导模型进行关系推理。

结果：在引用、超链接、社交、协购四大 TRN 基准上一致领先；更关键的是，仅在节点级任务训练即可零样本推广到边级与图级任务，展现超越跨域迁移的结构性泛化。

意义：把 R1-Zero 的"纯 RL / 无 SFT"思想首次推广到图 + 文本混合模态，为知识图推理 RL 树立新基线。

NGC：在 "推理–遗忘"联合学习中控制 KV 缓存，纯 outcome 奖励驱动

Michael Y. Li, Jubayer Ibn Hamid, Emily B. Fox, Noah D. Goodman（斯坦福大学）
arXiv: 2604.18002 · 2026-04-20

核心贡献：CoT 推理每步都扩张 KV 缓存，是长推理继续 scale 的根本瓶颈。提出 Neural Garbage Collection (NGC)——让语言模型在推理同时学习遗忘：周期性暂停、决定驱逐哪些 KV 条目、然后在剩余缓存上继续推理。整个过程仅用 outcome-based 任务奖励端到端训练。

关键设计：(i) 驱逐决策被形式化为稀疏离散动作；(ii) 推理-遗忘交替被实现为 token 流上的状态机；(iii) 奖励只来自最终答案正确性，不需要逐条缓存的标签——模型必须自己发现哪些信息值得保留。

结果：在数学与多跳推理任务上，NGC 缩减 ~50% KV 缓存的同时准确率无下降、部分超越全缓存基线。展示了"学习遗忘"作为 RL 目标的可行性。

意义：与 KV eviction/quantization 研究脱钩，首次给出"reward-driven forgetting"作为推理效率 RL 新范式。

POP：基于预训练文本的 rubric-based Self-Play，把 RL 自博弈扩展到开放域

Chengyu Huang 等（4 名作者）
arXiv: 2604.20051 · 2026-04-22

核心贡献：Self-Play 在数学与代码已有显著效果，但开放域因缺乏可验证奖励长期无法受益。提出 POP (Post-trains on Open-ended tasks via Pretraining)——同一 LLM 同时合成"查询 + 回答 + 评估 rubric"，再用 rubric 打分，DPO 训练模型。

防 reward hacking 关键：(1) 特权预训练文本——让 rubric 生成过程可以"偷看"预训练语料，保留生成-验证差距；(2) 只取得分最高与最低的 pair 做 DPO，避开中间高方差段。

结果：在 Qwen-2.5-7B 上提升多项开放评测；理论上首次给出"rubric 自博弈"避免崩溃的充分条件。

意义：与 TCER（2604.11522）"开放写作 RLVR"并列，共同把 RL 从 verifiable 走向 evaluable 的下一里程碑。

DCF：让 Conformal Factuality 成为可微训练目标

Nathan Hittesdorf 等（3 名作者）
arXiv: 2604.20098 · 2026-04-22

核心贡献：Conformal Factuality 以依赖图表示输出并与"逻辑先祖"联合验证，能给 LLM 事实性提供严格的统计保证；但原算法不可微，必须手工打分，从而在高可靠水平下删除高达 60% 的真声明。提出 Differentiable Coherent Factuality (DCF)：对原算法做完全可微松弛，既保留统计保证又可学习更好打分器。

方法：把依赖图上的联合验证过程重写为可微的概率积分；引入"温度退火"收紧与原算法的等价性。

结果：在两个推理基准上，真声明保留率 +141%，同时维持原有的 hallucination rate 控制。为 LLM 可靠性设立"带保证的 RL 训练目标"。

意义：打通 Conformal Prediction 与 RL post-training 之间的"可微墙"，是近期 Factuality RL 最重要的单点突破。

IV. 奖励建模 · Reward Modeling & Alignment

R2IF：把 CoT 有效性量化为 RL 奖励，端到端优化函数调用

Aijia Cheng, Kailong Wang, Ling Shi, Yongxin Zhao
arXiv: 2604.20316 · 2026-04-22

核心贡献：现有函数调用 RL 方法普遍存在"推理过程与实际调用决策不一致"。提出 R2IF (Reasoning-to-Interpretable Function calling)——一个 GRPO-based 复合奖励框架：
• 格式/正确性奖励；
• Chain-of-Thought Effectiveness Reward (CER)：量化 CoT 对最终调用决策的边际贡献；
• Specification-Modification-Value (SMV) 奖励：分拆参数规范、修改与取值三个语义维度。

结果：在 BFCL/ACEBench 上，Llama3.2-3B 相对基线最高 +34.62%；平均 CoT Effectiveness 变为正值（0.05），证明 CoT 确实在辅助决策而非装饰。

意义：把"CoT 是否真的有用"从人工评估升级为可微奖励，是可解释工具调用方向的第一个量化指标。

Beyond Overlap Metrics：GRPO + 双重奖励提升多角色对话摘要忠实度

作者信息待公开
arXiv: 2604.17188 · 2026-04-19

核心贡献：多角色对话摘要现有方法一味优化 ROUGE/BERTScore，导致表面模仿而非忠实。提出认知式推理 + 奖励优化的两阶段框架：
① 从教师大模型蒸馏结构化推理轨迹，用作辅助监督初始化"推理感知摘要器"；

意义：给"摘要 RLVR"提供一套可落地的结构化奖励模板，缓解摘要任务长期"指标驱动 → 漂移"困境。

V. 基准与环境 · Benchmarks & Training Environments

SkillLearnBench：首个评估 LLM Agent 持续技能学习的基准

Shanshan Zhong, Yi Lu, Jingjie Ning, Yibing Wan, Lihan Feng, Yuyi Ao, Leonardo F. R. Ribeiro, Markus Dreyer, Sean Ammirati, Chenyan Xiong（卡内基梅隆大学 & Amazon AGI）
arXiv: 2604.20087 · 2026-04-22

核心贡献：技能已成为让 LLM Agent 在定制指令 / 工作流 / 工具下完成复杂真实任务的主流方式，但"如何自动、持续地学习这些技能"仍缺乏评估。SkillLearnBench 是首个面向连续技能学习方法的基准——包含 20 个经验证的技能依赖任务，覆盖真实技能分类学中的 15 个子领域。

三级评估：(i) 技能质量；(ii) 执行轨迹；(iii) 任务结果。这种设计区分了"学会技能 vs 正确使用技能 vs 技能是否真有帮助"三个层面。

意义：为 Agent-RL 与 continual learning 提供长期缺失的评估基础，和 ClawEnvKit（2604.18543）、FT-Bench（2604.14116）共同构筑 Agentic RL 的"公共靶场"。

arXiv: 2604.20601 · 2026-04-22

意义：把"规划者 = 教师, 执行者 = 学生"的固化角色打破为互为教师的闭环，为 goal-conditioned RL 提供新范式。
